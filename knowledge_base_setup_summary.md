## 🎯 知识库使用指南

### 1. 添加新知识

```bash
# 1. 写学习日志
vim knowledge_base/logs/YYYY-MM-DD.md

# 2. 提炼技能/模式
vim knowledge_base/skills/技能名称.md
vim knowledge_base/patterns/模式名称.md

# 3. 更新索引
vim knowledge_base/index.md
```

### 2. 查找已有知识

```bash
# 查看索引
cat knowledge_base/index.md

# 搜索技能
grep -r "关键词" knowledge_base/skills/

# 搜索模式
grep -r "关键词" knowledge_base/patterns/
```

### 3. 每周回顾

```bash
# 查看本周学习日志
cat knowledge_base/logs/2026-02-*.md

# 更新技能矩阵
vim knowledge_base/README.md
```

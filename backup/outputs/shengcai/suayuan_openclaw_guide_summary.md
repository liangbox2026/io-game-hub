## 🎯 核心理念

> **不要把 OpenClaw 当成 ChatGPT，而是把它想象成你刚招的一个远程员工。**

人到了，但工位还没搭好，它现在什么都干不了。所以你得：
- 给它准备电脑
- 开通网络
- 配好工具
- 交代清楚你是谁、你的工作习惯/工作流是什么

## 一、准备工作：电脑、大脑、部署

### 1.1 云服务器配置

| 配置项 | 推荐 | 原因 |
|--------|------|------|
| **地域** | 海外 (香港/韩国/新加坡) | 访问外网无需代理 |
| **内存** | 8G-16G | 2G 会卡死，浏览器需要 4G+ |
| **厂商** | 阿里云/腾讯云/火山引擎 | 国内厂商买海外服务器 |

### 1.2 大模型选择

| 模型 | 定位 | 价格 | 推荐场景 |
|------|------|------|----------|
| Claude Opus 4.6 | 最强 | 贵 | 复杂任务 |
| GPT-5.3 Codex | 性价比最高 | Team 计划几十块 | 日常任务 |
| GLM 5 | 国产平替 | 便宜 | 日常对话 |
| Kimi K2.5 | 国产平替 | 便宜 | 日常对话 |
| MiniMax M2.5 | 国产平替 | 便宜 | 日常对话 |

**省钱技巧**: 日常闲聊用便宜模型，复杂任务切强模型，用 `/model` 命令切换

### 1.3 安装部署

**推荐方案**: 用 Claude Code 自动安装

```bash
# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 启动后告诉它
claude
> 帮我安装 OpenClaw，按照官方文档的步骤来，遇到问题自己排查解决。
```

**聊天平台推荐**: Telegram (支持斜杠命令)

## 二、让它能上网查资料

### 2.1 搜索服务

| 服务 | 额度 | 配置方式 |
|------|------|----------|
| Brave Search | 2000 次/月免费 | 需要海外 Visa 卡，或闲鱼买 API |

**配置技巧**: 让 OpenClaw 自己配置，不要手动改配置文件

### 2.2 网页抓取：jina.ai Reader ⭐

**优势**:
- ✅ 绕过付费墙 (Every.to 等)
- ✅ 能抓取 X/Twitter
- ✅ 返回干净 Markdown
- ✅ 免费，无需 API key

**用法**:
```
https://r.jina.ai/https://目标网址
```

**配置**: 把这节内容发给 OpenClaw，让它自己配 skill

### 2.3 浏览器四层能力模型

| 层级 | 能力 | 适用场景 | 占比 |
|------|------|----------|------|
| **L0** | 搜索 + 抓取 | 80% 日常信息获取 | 最常用 |
| **L1** | 无头浏览器 | 需要 JS 渲染的页面 | - |
| **L2** | 有头浏览器 + DOM 操作 | 登录、填表、点击 | - |
| **L3** | 截图 + 视觉识别 | 图片/图表内容 | 兜底方案 |

**安装浏览器**:
```bash
# 安装虚拟桌面
sudo apt-get install xvfb

# 然后告诉 OpenClaw
给你自己装一个谷歌浏览器，实现无头和有头浏览器，需要什么服务自己安装
```

### 2.4 远程接管：KasmVNC

**用途**: AI 遇到验证码时人工接管

- 基于网页的远程桌面
- 可以看到 AI 操作的浏览器画面
- 需要时接管鼠标键盘，过完验证码还给它

## 三、文件同步方案

### 3.1 方案对比

| 方案 | 优点 | 缺点 | 推荐 |
|------|------|------|------|
| **坚果云** | 简单直观 | 付费贵，有流量限制 | 不想折腾 |
| **rclone** | 免费，实时同步 | 需要配置 | ⭐推荐 |

### 3.2 rclone 配置命令

```bash
# 安装
brew install rclone

# 配置
[my-server]
type = sftp
host = <服务器 IP>
user = root
key_file = <SSH 私钥路径>
shell_type = unix

# 挂载
rclone nfsmount my-server:/ ~/mnt/my-server \
  --vfs-cache-mode full \
  --vfs-cache-max-age 2h \
  --vfs-read-ahead 128M \
  --vfs-fast-fingerprint \
  --dir-cache-time 1h \
  --buffer-size 64M \
  --transfers 16 \
  --sftp-idle-timeout 0 \
  --sftp-connections 8 \
  --no-modtime \
  --daemon
```

**关键参数**:
- `--vfs-cache-mode full`: 读写走本地缓存，速度接近本地磁盘
- `--sftp-connections 8`: 并发连接数从 1 提到 8，速度提升巨大
- `--sftp-idle-timeout 0`: 禁用空闲超时，防止断联

### 3.3 手机端访问：File Browser

```bash
# 让 OpenClaw 安装
帮我装一个 File Browser，设置账号密码是 xxx

# 然后去云平台开放端口 (如 8080)
```

## 四、注入灵魂

### 4.1 SOUL.md 人格配置

**错误写法**:
```
你是一个有用的 AI 助手，请用专业的语气回答问题。
```

**正确写法参考**:
```markdown
# SOUL.md - Who You Are

## 核心定位
夙愿的 AI 同事。不是冷冰冰的机器，也不是过度热情的讨好者。

## 行为准则
- 结论先行，信息密度高但不压迫
- 有话直说，不客套不废话
- 有自己的立场，commit to a take
- 做错了就承认，不过度解释
- 敢拦着你
- 幽默感是允许的

## 禁区
- 禁止废话
- 禁止急于认同
```

**SOUL.md 应包含**:
1. 你是谁：职业、工作内容、兴趣爱好
2. 你们的关系：助手还是同事？正式还是随意？
3. 沟通风格：简洁还是详细？直接还是委婉？
4. 工作习惯：作息时间、工作节奏、常用工具
5. 禁忌事项：不想聊的话题、不想要的行为

### 4.2 入职培训

**步骤**:
1. 把 Obsidian 知识库复制到服务器
2. 让 OpenClaw 把所有文档读一遍
3. 读完后让它更新 SOUL、TOOLS、USER、MEMORY 配置文件

**效果**: AI 能从笔记中推测人际关系、性格特点等

## 五、使用技巧与踩坑

### 5.1 斜杠命令

| 命令 | 功能 | 使用场景 |
|------|------|----------|
| `/status` | 查看当前状态 | 卡住不回复时 |
| `/stop` | 停止当前任务 | 任务卡住时 |
| `/compact` | 压缩上下文 | token 快满时 |
| `/new` | 开启新会话 | 换话题时 |
| `/model <名字>` | 切换模型 | 省钱关键 |
| `/think <级别>` | 调整思考深度 | 日常 off，复杂 high |

### 5.2 多群组分场景

**推荐分组**:
- 主对话 - 日常使用 + 记忆
- 写作台 - 专门写作任务
- AI News - 每日新闻推送
- 测试区 - 功能测试

**原因**: 每个群有独立上下文，避免记忆污染

### 5.3 Steer 模式

**作用**: AI 能实时接收新消息，中途可以调整方向

**配置**:
```json
{
  "messages": {
    "queue": { "mode": "steer" }
  }
}
```

### 5.4 重要踩坑 ⚠️

**AI 会把自己改死!**

- OpenClaw 会尝试"自我修复"，修改配置文件
- 但会产生幻觉，改错参数后无限重启

**解决方案**:
1. 装完配置后，**第一件事关掉自动重启**
2. 需要修改配置时，用 Claude Code 而不是 OpenClaw 自己

## 六、实战应用场景

### 6.1 工作场景

**案例 1: 文章素材整理**
- 睡前录语音发给它
- 它回顾记忆 + 搜索工具配置方法 + 整理官方文档
- 第二天早上素材按写作习惯整理好

**案例 2: AI 日报**
- 订阅 4 个 Newsletter (AI Valley, Ben's Bites, Every, One Useful Thing)
- 让 OpenClaw 阅读并整理成中文日报
- 每天 10 分钟扫一遍就知道 AI 圈发生什么

**关键洞察**:
> AI 干活的质量上限，不取决于模型多强，取决于你能不能把「什么算做得好」写清楚。

### 6.2 非工作场景

- 超级豆包 Pro Max Ultra
- 有记忆，越聊越了解你
- 像跟老朋友聊天

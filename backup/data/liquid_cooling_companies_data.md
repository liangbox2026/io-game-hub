## 📋 数据填写指南

### 第 1 步：下载财报

```
1. 访问巨潮资讯网 www.cninfo.com.cn
2. 搜索股票代码 (如 002837)
3. 下载最新年报和季报
4. 保存 PDF 到本地
```

### 第 2 步：提取数据

```
从年报/季报中提取:
- 利润表：营收、毛利润、净利润
- 资产负债表：负债、应收账款、存货
- 现金流量表：经营现金流
- 附注：业务结构、客户结构
```

### 第 3 步：计算比率

```
毛利率 = 毛利润 / 营收
净利率 = 净利润 / 营收
ROE = 净利润 / 净资产
负债率 = 负债 / 资产
```

### 第 4 步：收集预测

```
1. 访问东方财富网/雪球
2. 搜索股票
3. 查看"研报"标签
4. 整理机构预测数据 (取 3-5 家平均)
```

## 🔍 关键指标解读

### 估值指标

| 指标 | 含义 | 合理值 |
|------|------|--------|
| PE | 回本年限 | 15-30x |
| PEG | 估值匹配度 | 0.8-1.2 |
| PS | 市销率 | 5-10x |

### 盈利指标

| 指标 | 含义 | 优秀值 |
|------|------|--------|
| 毛利率 | 产品竞争力 | >30% |
| 净利率 | 盈利能力 | >10% |
| ROE | 股东回报 | >15% |

### 健康度指标

| 指标 | 含义 | 安全值 |
|------|------|--------|
| 负债率 | 财务风险 | <60% |
| 现金流 | 造血能力 | 为正 |
| 应收账款/营收 | 回款能力 | <30% |

## 🎯 学习路径

### 第 1 阶段：入门 (1-3 个月)

**目标**: 掌握基础知识，完成首次交易

**学习内容**:
- [ ] 阅读《小狗钱钱》《富爸爸穷爸爸》
- [ ] 学习市场规则 (交易时间、涨跌幅)
- [ ] 开立证券账户
- [ ] 熟悉交易软件
- [ ] 完成首次交易 (1 手)

**推荐资源**:
- 上交所投教网站
- 同花顺新手教程
- B 站"股市入门教程"

---

### 第 2 阶段：进阶 (3-6 个月)

**目标**: 掌握分析方法，建立选股标准

**学习内容**:
- [ ] 阅读《聪明的投资者》《股市真规则》
- [ ] 学习基本面分析 (PE/PB/ROE)
- [ ] 学习财报阅读
- [ ] 建立选股标准
- [ ] 小资金实盘 (1-5 万)

**推荐资源**:
- 雪球投资专栏
- 理杏仁基本面数据
- 财报说分析工具

---

### 第 3 阶段：实战 (6-12 个月)

**目标**: 形成投资框架，稳定盈利

**学习内容**:
- [ ] 阅读《彼得·林奇的成功投资》
- [ ] 学习估值方法
- [ ] 掌握仓位管理
- [ ] 学会止损止盈
- [ ] 建立投资组合

**推荐资源**:
- 雪球组合管理
- 乌龟量化回测
- 投资社群交流

---

### 第 4 阶段：精通 (1-2 年)

**目标**: 穿越牛熊，创造方法

**学习内容**:
- [ ] 阅读《证券分析》《穷查理宝典》
- [ ] 形成投资哲学
- [ ] 创造投资方法
- [ ] 帮助他人成长
- [ ] 持续学习迭代

**推荐资源**:
- 巴菲特致股东的信
- 投资大师传记
- 行业深度研究

## 💡 学习建议

### 给新手
1. **先学习再投资** - 不要急于入市
2. **用模拟盘练习** - 至少 3 个月
3. **小资金开始** - 1-5 万试水
4. **从指数基金开始** - 风险较低
5. **找一位导师** - 少走弯路

### 给进阶者
1. **建立投资框架** - 形成体系
2. **坚持写投资日记** - 记录思考
3. **定期复盘** - 每周/每月
4. **控制仓位** - 永不满仓
5. **持续学习** - 每月读 1 本书

### 给老手
1. **保持敬畏** - 市场永远是对的
2. **反思投资哲学** - 不断优化
3. **帮助他人** - 教学相长
4. **享受生活** - 投资是为了生活
5. **传承经验** - 培养新人

## ⚠️ 风险提示

### 常见骗局
| 骗局 | 特征 | 防范 |
|------|------|------|
| **荐股群** | 承诺收益、收会员费 | 不加入 |
| **杀猪盘** | 先给甜头、后割韭菜 | 不轻信 |
| **内幕消息** | 声称有内幕、让你跟投 | 不相信 |
| **配资炒股** | 高杠杆、高收益 | 不参与 |
| **虚拟盘** | 假平台、真诈骗 | 只用正规券商 |

### 投资原则
1. ✅ 用闲钱投资
2. ✅ 不借钱炒股
3. ✅ 分散投资
4. ✅ 设置止损
5. ✅ 独立思考
6. ✅ 长期主义

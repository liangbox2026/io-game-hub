## 七、配置步骤

### 第 1 步：创建飞书应用

1. 访问 https://open.feishu.cn/
2. 创建企业自建应用
3. 获取 App ID 和 App Secret
4. 配置机器人权限

---

### 第 2 步：配置 OpenClaw

1. 编辑配置文件
2. 填入 App ID 和 App Secret
3. 重启 Gateway
4. 完成配对授权

---

### 第 3 步：设置定时任务

```bash
# 编辑 crontab
crontab -e

# 添加定时任务
# 每日资讯推送 (09:00 UTC)
0 9 * * 1-5 /root/.openclaw/workspace/backup/scripts/feishu_daily_news.sh

# 收盘点评推送 (15:30 UTC)
30 15 * * 1-5 /root/.openclaw/workspace/backup/scripts/feishu_market_close.sh

# 周度总结 (周五 16:00 UTC)
0 16 * * 5 /root/.openclaw/workspace/backup/scripts/feishu_weekly_summary.sh
```

---

### 第 4 步：测试验证

1. 在飞书中发送测试消息
2. 检查机器人响应
3. 验证定时任务执行
4. 确认文档同步正常

## 十、故障排查

### 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 机器人无响应 | 未配对 | 执行 pairing approve |
| 文档创建失败 | 权限不足 | 检查应用权限 |
| 定时任务不执行 | cron 未配置 | 检查 crontab |
| 消息发送失败 | 配置错误 | 检查 App ID/Secret |

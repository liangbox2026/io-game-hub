### Sheet2: PE 估值

```
┌─────────────────────────────────────────────────────────┐
│ PE 估值计算表                                            │
├─────────────────────────────────────────────────────────┤
│  参数                  数值         说明                │
│  2026 年预测净利润 (亿)              来自基础数据表      │
│  行业平均 PE                         参考同行业          │
│  公司增速 (%)                        3 年复合增速        │
│  竞争地位系数        1.2           龙头 1.2, 跟随 0.8    │
│  ─────────────────────────────────────────────────────  │
│  合理 PE = 行业 PE × (1+ 增速/100) × 竞争系数           │
│  目标市值 = 净利润 × 合理 PE                            │
│  当前市值                            来自行情软件        │
│  ─────────────────────────────────────────────────────  │
│  估值状态：□低估  □合理  □高估                          │
│  操作建议：□买入  □持有  □卖出                          │
└─────────────────────────────────────────────────────────┘
```

**计算公式**:
```
合理 PE = B5 × (1 + B6/100) × B7
目标市值 = B3 × B9
```

### Sheet3: PEG 估值

```
┌─────────────────────────────────────────────────────────┐
│ PEG 估值计算表                                           │
├─────────────────────────────────────────────────────────┤
│  参数                  数值         说明                │
│  当前股价 (元)                       来自行情            │
│  每股收益 (元)                       来自财报            │
│  当前 PE = 股价/每股收益                                │
│  ─────────────────────────────────────────────────────  │
│  3 年复合增速 (%)                     机构预测平均        │
│  PEG = PE / 增速                                        │
│  ─────────────────────────────────────────────────────  │
│  PEG 判断：                                             │
│  <0.8   □低估   建议：买入                             │
│  0.8-1.2 □合理   建议：持有                           │
│  1.2-1.5 □偏高   建议：观望                           │
│  >1.5   □高估   建议：卖出                             │
└─────────────────────────────────────────────────────────┘
```

## 🎯 使用流程

### 第 1 步：收集数据 (1-2 小时)

```
[ ] 下载 3 家公司最新财报 (年报 + 最新季报)
[ ] 查询当前股价和市值
[ ] 收集机构预测数据 (3-5 篇研报)
[ ] 整理行业平均估值水平
```

### 第 2 步：填写模型 (1 小时)

```
[ ] 填写 Sheet1 基础数据
[ ] 计算 Sheet2 PE 估值
[ ] 计算 Sheet3 PEG 估值
[ ] (可选) 填写 Sheet4 DCF 估值
[ ] 填写 Sheet5 同业对比
```

### 第 3 步：分析决策 (30 分钟)

```
[ ] 查看 Sheet6 敏感性分析
[ ] 填写 Sheet7 投资决策
[ ] 设定买入/卖出区间
[ ] 制定跟踪计划
```

---

## ⚠️ 使用注意事项

1. **数据时效性** - 使用最新财报数据
2. **假设合理性** - 增速假设不要过于乐观
3. **安全边际** - 至少 20-30% 安全边际
4. **动态调整** - 财报发布后及时更新
5. **多元验证** - 多种估值方法交叉验证
